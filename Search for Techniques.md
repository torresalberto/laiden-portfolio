# Photographer Website Development

**Project Context**
The objective is to build a website for a photographer.

**Initial Steps and Resources**
*   **Photo Acquisition:** Download the images from the photographer's Instagram (the URL will be provided). It is necessary to download between fifty and one hundred photos, if possible.
*   **Project Folder:** Locate the project development folder (it will also be provided).
*   **Tools:** All necessary tools are already available, as similar projects have been completed previously.

**Technology Stack (AI Models)**
Two AI models will be used:
*   **MinimAx 2.5:** Code model, will act as a code engineer.
*   **MiMo B2 Omni:** AI model with vision capabilities, responsible for photo analysis.

**Roles and Workflow**
1.  **MinimAx 2.5** will initiate the process by downloading all photos.
2.  **MiMo B2 Omni** will analyze the photos, provide creative analyses, and save these analyses along with instructions.
3.  **MinimAx 2.5** will use MiMo's analyses and instructions to build the website.
4.  The models should communicate with each other, possibly via a file.

**Website Vision and Requirements**
*   The website should be a visual extension of the images.
*   It should reflect the patterns, colors, and inherent style of the photographs.
*   The website's narrative will be built **solely with images**, without using text.
*   **Contact:** The only point of contact on the website will be a link to the photographer's Instagram, configured as a "send message" button to direct traffic to their profile.

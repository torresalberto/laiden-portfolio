## 0. THE CORE PHILOSOPHY

**Laiden's work lives in the tension between warmth and coldness.**

21 images radiate warmth (amber skin tones, intimate portraits, ember-lit studios).
18 images breathe cold (blue nights, teal atmospheres, solitary environments).

The website is not a gallery. It is a **dark room**. Each photograph is a single light source. The darkness is not empty — it is the canvas that makes the light matter.

The design must be **invisible**. No text. No navigation. No chrome. Just the photographs and the void between them.

### 1.2 Warm Palette (21 images, avg warmth 2.11, avg brightness 63.2)

Dominant colors: `#000000`, `#202000`, `#200000`, `#402000`, `#202020`

These images glow. They are amber-lit, skin-toned, intimate. The warmth comes from:
- Studio lighting with warm gels
- Golden hour natural light
- Skin tones against dark backgrounds
- Amber/terracotta environmental elements

### 1.3 Cool Palette (18 images, avg warmth 0.40, avg brightness 43.4)

Dominant colors: `#000000`, `#000020`, `#002020`, `#002040`, `#004040`

These images breathe cold. They are blue-drenched, atmospheric, solitary. The coolness comes from:
- Night photography
- Blue/teal color grading
- Urban environments at dusk
- Water and sky elements

### 1.4 The Bimodal Tension

**This is the defining characteristic.** The collection oscillates between two poles:
- Warm (r/b ratio > 1.0): 21 images, avg brightness 63.2
- Cool (r/b ratio < 1.0): 18 images, avg brightness 43.4

Cool images are darker (43.4 vs 63.2 brightness). This creates a natural rhythm — warm images feel like relief after cold ones.

**Design Implication:** The background stays `#0a0a0a` always. The transition between warm and cool images IS the emotional experience.

## 5. WEBSITE DESIGN SYSTEM

### 5.1 Layout: Cinematic Full-Screen Experience

**Decision rationale:**
- Portrait images (39%) need vertical space to breathe
- Landscape images (61%) need horizontal space for cinematic impact
- Mixed orientations rule out grid/masonry — each image demands full viewport
- The darkness between images is part of the experience

**Visual Structure:**

### 5.3 Animation System

#### 5.3.1 Image Transitions
- **Type:** Cross-fade (opacity overlay)
- **Duration:** 1200ms
- **Easing:** `cubic-bezier(0.4, 0, 0.2, 1)`
- **Behavior:** Outgoing image fades to 0, incoming fades to 1 simultaneously
- **NO slide, NO zoom-in transition, NO parallax**

#### 5.3.2 Ken Burns Effect (on active image only)
- **Type:** Subtle scale animation
- **Scale:** 1.0 → 1.015 (1.5% zoom over 10 seconds)
- **Easing:** `cubic-bezier(0.37, 0, 0.63, 1)` (breathe)
- **Direction:** Always zoom in, never out
- **Reset:** Scale resets to 1.0 on image change, then animates again
- **Purpose:** Gives still images a living, breathing quality

#### 5.3.3 Progress Dots
- **Default:** 5px circle, `rgba(255,255,255,0.2)`
- **Active:** 7px circle, `rgba(255,255,255,0.5)`
- **Transition:** 300ms ease
- **Glow:** Active dot has `box-shadow: 0 0 8px rgba(255,255,255,0.15)`
- **Hover:** 5px → 6px, `rgba(255,255,255,0.35)`
- **Click:** Navigate to that image

#### 5.3.4 Instagram Button
- **Default:** 48px circle, `rgba(255,255,255,0.08)` background, `1px solid rgba(255,255,255,0.12)` border
- **Icon:** SVG envelope/message, `rgba(255,255,255,0.5)` stroke
- **Hover:** Background `#c4785a`, border `#c4785a`, `box-shadow: 0 0 24px rgba(196,120,90,0.4)`, icon `#fff`
- **Transition:** 400ms `cubic-bezier(0.4, 0, 0.2, 1)`
- **Scale on hover:** 1.0 → 1.08
- **Mobile:** 56px circle

### 5.4 Interaction Model

#### Desktop
| Input | Action |
|-------|--------|
| Click right 60% of screen | Next image |
| Click left 40% of screen | Previous image |
| Arrow Right / Space | Next image |
| Arrow Left | Previous image |
| Home | First image |
| End | Last image |
| Hover on image | Pause auto-advance |
| Mouse leave image | Resume auto-advance |
| Scroll down | Next image (debounced 800ms) |
| Scroll up | Previous image (debounced 800ms) |

#### Mobile/Touch
| Input | Action |
|-------|--------|
| Swipe left (>50px) | Next image |
| Swipe right (>50px) | Previous image |
| Tap right 60% | Next image |
| Tap left 40% | Previous image |
| Tap progress dot | Navigate to that image |
| Tap Instagram button | Open Instagram |

#### Auto-Advance
- **Interval:** 6000ms (6 seconds — slow, contemplative)
- **Pause on:** Hover, touch start, keyboard interaction
- **Resume on:** Mouse leave, 3000ms after last interaction
- **Loop:** After last image, return to first (seamless)

### 5.5 Loading Strategy

#### Initial Load
1. **First image:** Inline preload (display immediately)
2. **Next 2 images:** Preload via `<link rel="preload">`
3. **Remaining 36 images:** Lazy load on demand

#### During Navigation
- **On image N:** Preload N+1 and N+2 (wrap around at end)
- **While transitioning:** Show `#0a0a0a` background (no loading spinner)
- **If image fails:** Show solid `#1a1a1a` with no error message

#### Performance Targets
- **First paint:** <500ms (first image is critical)
- **Interactive:** <1s
- **Total page weight:** ~5.7 MB (all 39 images)
- **Initial download:** ~300 KB (HTML + first 3 images)

### 5.6 Responsive Breakpoints

| Breakpoint | Width | Behavior |
|-----------|-------|----------|
| Desktop | >1024px | Full viewport, centered image, 48px contact button |
| Tablet | 768-1024px | Full viewport, image fills width, 48px contact button |
| Mobile | <768px | Full viewport, image fills screen, 56px contact button, dots smaller |

**Mobile-specific:**
- Contact button: 56px (larger tap target)
- Progress dots: 4px default, 6px active
- Bottom padding: 60px (more space for thumb)
- Swipe threshold: 40px (easier on mobile)
- Auto-advance: 8000ms (slower on mobile — people scroll slower)

## 9. THE EXPERIENCE

When someone opens this website:

1. **First impression:** Darkness. Then a photograph emerges from it. No chrome. No text. Just the image.

2. **Navigation:** They click or swipe. The image dissolves into the next. The darkness between them is the breath.

3. **Rhythm:** Every 6 seconds, the next image appears. They can control the pace, but the default is slow — like turning pages in a photo book.

4. **Discovery:** The images alternate between warm portraits and cool landscapes. This is intentional — it's the photographer's visual language.

5. **Contact:** A subtle button in the corner. On hover, it glows warm terracotta. It links to Instagram DMs. That's the only way to reach the photographer.

6. **Feeling:** The website should feel like standing in a dark gallery. The photographs are lit one at a time. The darkness is not empty — it's the frame.
